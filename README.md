# pythonTraning
This is for someone who is starting to learn Python.

## Overview
Small Python examples for beginners:
- `variables.py`: variable declaration, types, assign/reassign, arithmetic
- `functions.py`: function definition, call, return values
- `control_flow.py`: if/else, loops (`for`,`while`,`break`,`continue`)
- `data_structures.py`: list, tuple, set, dict basics

## How to run
From terminal:
- `cd /Users/dletreda/Documents/Python/pythonTraning`
- `python variables.py`
- `python functions.py`
- `python control_flow.py`
- `python data_structures.py`

## Practice tasks
1. Edit `variables.py` to include `*`, `/`, `%`, and `**` operations.
2. In `functions.py`, add a new function `subtract(a, b)` and call it.
3. Modify `control_flow.py` to include `for i in range(1, 11)` with a `break` at 7.
4. In `data_structures.py`, add and print a new dictionary entry.

## Notes
- Use `print()` to inspect values.
- Comments begin with `#`.
- Save file and re-run commands as you change code.
